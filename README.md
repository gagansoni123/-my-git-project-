# My Git Project

## Project Overview
This project is designed to help users track their daily tasks efficiently. It demonstrates
core Git and GitHub workflows as part of a comprehensive assignment.


## Installation
1. Clone the repository:
   git clone https://github.com/yourusername/my-git-project.git
2. Install required dependencies (if any):
   pip install -r requirements.txt

## Features
- Add, edit, and delete tasks
- Mark tasks as completed
- View daily and weekly task summary


## Usage
1. Run the main program:
   python main.py
2. Follow the on-screen instructions to manage your tasks.
